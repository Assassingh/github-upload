Start.m: Base file. Has the code for reading the data and calls the other functions. Also has sections to run the wrapper and filter iteratively and for generating graphs and calculating accuracies.
forwardselection.m: Algorithm for the wrapper method of feature selection.
rankingfeat.m: Algorithm for selecting top k% features based on their AVR.
plotFeat.m: Generates plots of top 20 features. Unchanged.
split.m: Splits data. Unchanged.
